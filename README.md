# Scartellato 🤌

Benvenuti nel repository ufficiale di Scartellato, il linguaggio più napoletano che esista


🚀 Requisiti di Sistema
Per compilare e testare il progetto sul proprio sistema, l'unico prerequisito fondamentale è la presenza di un compilatore compatibile con il C:

GCC (GNU Compiler Collection) installato e configurato nel PATH di sistema.


# Clona la repository
git clone https://github.com/Re-1234/Scartellato.git

# Esecuzione dei Test
Il progetto include una suite completa di test per verificare la correttezza semantica, la gestione dello scope, le funzioni e i costrutti del linguaggio.

Per eseguire tutti i test e visualizzare l'esito direttamente nel terminale, è sufficiente avviare l'apposito script: run_test_cases che mostrerà le statistiche su un numero scelto di casi di test
